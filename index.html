HTML basico

HTML

Que es HTML?

HTML es el lenguage sobre el cual esta construida toda la web.

HTML son las siglas de Hyper Text Markup Language, es decir lenguaje de marcado de hipertexto.

Para comprender mejor esto vamos a desmenusar estas siglas.

Hypertext:

Se refire a un texto que contiene un enlace, es decir el texto plano en un texto que solamente puedes leerlo,

pero un hipertexto es aquel texto que contiene fragmentos donde tu puedes hacer click en un navegador grafico y eso te llevara a otro documento, esto permite que los documentos esten interconectados entre si.

Esto es algo muy elemental pero es la base de toda la web y en su momento fue una completa revolucion.

Markup:

Significa que html es un lenguaje que define la estructura de los documentos, es decir, en un documento indica que parte es el titulo, que parte es un parrado, que parte es una imagen, o que parte es un enlace, etc.

Language:

La palabra lenguaje nos indica que claramente HTML es un lenguaje, no un lenguaje de programacion pero si un lenguaje de marcado.

La web y HTML

La web entones es el conjunto de documentos HTML interconectados a traves de enlaces alrededor de todo el mundo,es por eso se llama World wide web , o red de alcance global, la world wide web funciona a traves de internet, por el cual se pueden transferir diversos tipos de datos a traves de el protocolo Protocolo de Transferencia de datos o HTTP (Hypertext Transfer Protocol), que son los enlaces a las paginas webs.

Breve historio de HTML pero muy importante

Tim Berners-Lee es un cientifico de la computacion britanico, conocido por ser el creado de la World Wide Web.Establecio la primera comunicacion entre un cliente y un servidor usando el protocolo http en diciembre de 1990
