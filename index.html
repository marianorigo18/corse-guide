html

What is html?

html el lengueje de marcado de hipertexto que nos permite dar una estructura al a nuetro sitio web.

css

What is css?

Este lengueje en cascada nos permite darle un estilo a todas nuestras aplicaciones webs, donde podremos darle colores, sombras, formas, delineados y todo lo que se nos ocurra al sitio web.

javascript

What is javascript?

Es el lengueje mas importante hasta la fecha, y el motor que se encargara de darle vida y dinamismo a nuestra aplicaciones, con ella aprenderemos:

dom

whats is dom?

El document objet model es la herramienta que nos pernitira acceder a las partes de nuestro sitio web, ya sea para cambiar algo o darle dinamismo desde javascript.

eventos

whats is events?

los eventos son acciones que activa el usuario para la interaccion con la apicacion, ya sea pulsando un boton, haciendo ckick etc,tambien veremos como enviar formularios..

json

what is json?

es el formato creado por javascript para poder enviar datos a traves de la red

localstorage

what is localstorage?

Es el metodo que nos permite almacenar informacion de el usuario en el explorador web, por ejemplo cuendo el usuario ya haya iniciado sesion y no lo tenga que volver a hacer.

Promesas: peticiones asincrnas

what is it?

Las peticiones asincronas son operaciones tareas que permiten ser procesadas sin detener la ejecucion normal del navegador, y esto se suele usar por ejemplo para obtener informacion del servidor.

funciones

what is funciones?

Las funciones son porciones de codigo que realizan una tarea en especial, y que nos permiten ejecutarlas n cantidad de veces sin estar repitiendo codigo

node js

what is node js?

es un entorno de trabajo backend de javascript orientado a eventos asincronos para la creacion de aplicaiones con interacciones a la base de datos.

mongodb

what is mongodb?

Mongodb es un sistema de bases de datos no relaciones, que nos permitira persistir datos y luego poder reutilizarlos un nuetras aplicaiones.

rest full api

what is rest full api?

Una rest api full es una interface que permite la comunicacion entre dos o mas sistemas para el seguro intercambio de datos a travez de los protocolos http
